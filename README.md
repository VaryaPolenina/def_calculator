# def_calculator (калькулятор)
изначально создаем функцию с приветствием, которая выводится один раз
далее идет функция со вторым приветсвием, для тех, кто в конце вычислений захочет зайти в калькулятор ещё раз
потом создаём саму функцию калькулятора, добавляем переменную, которая пригодится позже внутри программы, и переменную с вводом текста, где пользователю нужно выбрать, какую функцию он хочет использовать
далее идет проверка на if/else - ввёл ли пользователь доступную функцию или нет, если нет, то возвращаемся к строке ввода нужной функции
вводим бесконечный цикл 
в нём сначала проверяем ввёл ли пользователь функцию выхода, если да, то выходим, если нет, то продолжаем 
следующая проверка на корректно введённые числа через try/except - для этого изначально была введена переменная с False, если не корректно введено первое число, просим ввести заново, ели далее введено не корректно второе число, первое уже не трогаем и проим ввести корректно только второе число
дальше проверка на то, какую функцию ввёл пользователь через if/elif/else и в зависимости от этого выводим соответствующий результат вычислений
и в конце можно ещё раз воспользоваться калькулятором, нажав на любую клавишу
